## Quais módulos existem hoje?

Waitlist, Changelog, Roadmap, Feedback e Status.

## Como autentico a CLI?

```bash
cajuos login --apiKey caju_live_sk_xxx
```

## Posso integrar sem widget?

Sim. Use apenas API HTTP com `Authorization` e `X-Project-Id`.

## Os widgets funcionam em qualquer stack?

Sim. O SDK usa Web Components e funciona em HTML, Astro, Next.js, Nuxt e SvelteKit.

## Existe self-hosted oficial?

Hoje o produto é focado no modo cloud operado pelo time do cajuOS.
