# cajuOS Docs

![Banner cajuOS Docs](./public/cajuos-docs-readme-banner.png)

Documentação oficial do cajuOS.

Última atualização de conteúdo: **9 de março de 2026**.

## Acessar

- Site de docs (planejado): `https://docs.cajuos.dev`
- Código-fonte: este repositório

## Rodar localmente

```bash
npm install
npm run dev
```

## Contribuir

1. Edite os arquivos em `src/content/docs`.
2. Rode `npm run build` para validar.
3. Abra um PR.

## Deploy

O deploy de produção da docs é feito pelo time do cajuOS via pipeline oficial.
Quem contribui não precisa configurar infraestrutura de produção.
